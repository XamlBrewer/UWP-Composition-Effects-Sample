* Temperature and Tint (https://xamlbrewer.wordpress.com/2016/04/19/uwp-composition-effects-temperature-and-tint/)
* Effect chaining (https://xamlbrewer.wordpress.com/2016/05/15/uwp-composition-effects-chaining/)
